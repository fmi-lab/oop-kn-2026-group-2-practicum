# Жизнен цикъл на обектите, static, const

## 1) 
Създайте клас `Agent`, който има статичен метод `getCount()`, който връща броя на съществуващите обекти от този клас по време на извикването му.

```cpp
int main() {

    std::cout << Agent::getCount() << std::endl; // 0

    Agent smith, *jones;

    std::cout << Agent::getCount() << std::endl; // 1

    {
        Agent smith;
        jones = new Agent;
        std::cout << Agent::getCount() << std::endl; // 3
    }

    std::cout << Agent::getCount() << std::endl; // 2

    delete jones;

    std::cout << Agent::getCount() << std::endl; // 1

    return 0;
}
```

## 2)
Създайте клас `Dipper`, който има константен метод `getNumber()`, който връща уникален номер за всеки обект от този клас. Номерът трябва да бъде автоматично генериран при създаването на обекта и да не се променя през целия му жизнен цикъл.

```cpp
int main() {

    Dipper dipper1, dipper2;

    std::cout << dipper1.getNumber() << std::endl; // 1
    std::cout << dipper2.getNumber() << std::endl; // 2

    {
        Dipper dipper3;
        std::cout << dipper3.getNumber() << std::endl; // 3
    }
    Dipper dipper3;
    std::cout << dipper3.getNumber() << std::endl; // 4

    return 0;
}
```

## 3)
Създайте клас `Str`, който се грижи за динамично заделен масив от символи. Създайте конструктор, който приема `const char*` и заделя необходимата памет, за да съхрани този низ. 
- Създайте константен метод `c_str()`, който връща указател към този низ. Уверете се, че класът правилно освобождава заделената памет в деструктора си. 
- Създайте копиращ конструктор и оператор за присвояване, за да гарантирате правилното копиране на обектите от този клас.
- Създайте метод `length()`, който връща дължината на низа, без да включва терминалния нулев символ.
- Създайте статичен метод `cat(const Str& str1, const Str& str2)`, който връща нов обект от тип `Str`, представляващ конкатенацията на двата низа.
- Създайте статичен метод `cmp(const Str& str1, const Str& str2)`, който връща резултата от `strcmp` между двата низа.
- Създайте метод `cat(const Str& other)`, който конкатенира текущия низ с този на друг обект от тип `Str` и връща нов обект от тип `Str` с резултата.
- Създайте метод `cmp(const Str& other)`, който връща резултата от `strcmp` между текущия низ и този на друг обект от тип `Str`.

```cpp
int main() {

    Str str1("Hello"), str2("World");

    std::cout << str1.c_str() << " " << str2.c_str() << std::endl; // Hello World

    std::cout << Str::cat(str1, str2).c_str() << std::endl; // HelloWorld

    std::cout << Str::cmp(str1, str2) << std::endl; // <0

    str1 = str1.cat(str2);
    std::cout << str1.c_str() << std::endl; // HelloWorld

    std::cout << str1.cmp(str2) << std::endl; // <0

    return 0;
}
```

## 4)
Създайте клас `Matrix`, който представлява двумерен масив от числа. Създайте конструктор, който приема броя на редовете и колоните и заделя необходимата памет за съхранение на тези числа. Създайте деструктор, който освобождава заделената памет.

- Помислете как може да си улесните живота при заделянето и освобождаването на паметта.
- Ако някоя от следващите операции не може да бъде изпълнена поради несъвместими размери на матриците, извикайте `std::exit(1)`, като преди това отпечатате подходящо съобщение за грешка на стандартния изход.
- създайте метод `setValue(int row, int col, double value)`, който задава стойност на определена позиция в матрицата.
- създайте метод `getValue(int row, int col)`, който връща стойността на определена позиция в матрицата.
- създайте метод `at(int row, int col)`, който връща референция към стойността на определена позиция в матрицата, позволявайки директна промяна на стойността.
- създайте метод `print()`, който отпечатва съдържанието на матрицата във форматиран вид.
- създайте статичен метод `identity(int size)`, който връща нов обект от тип `Matrix`, представляващ единична матрица с даден размер.
- създайте статичен метод `add(const Matrix& mat1, const Matrix& mat2)`, който връща нов обект от тип `Matrix`, представляващ сумата на двете матрици. Уверете се, че двете матрици имат същите размери преди да извършите операцията.
- създайте статичен метод `multiply(const Matrix& mat1, const Matrix& mat2)`, който връща нов обект от тип `Matrix`, представляващ произведението на двете матрици. Уверете се, че броят на колоните в първата матрица е равен на броя на редовете във втората матрица преди да извършите операцията.

```cpp
int main() {

    Matrix mat1(2, 3), mat2(2, 3);

    mat1.setValue(0, 0, 1.0);
    mat1.setValue(0, 1, 2.0);
    mat1.setValue(0, 2, 3.0);
    mat1.setValue(1, 0, 4.0);
    mat1.setValue(1, 1, 5.0);
    mat1.setValue(1, 2, 6.0);

    mat2.setValue(0, 0, 7.0);
    mat2.setValue(0, 1, 8.0);
    mat2.setValue(0, 2, 9.0);
    mat2.setValue(1, 0, 10.0);
    mat2.setValue(1, 1, 11.0);
    mat2.setValue(1, 2, 12.0);

    std::cout << "Matrix 1:" << std::endl;
    mat1.print();

    std::cout << "Matrix 2:" << std::endl;
    mat2.print();

    Matrix sum = Matrix::add(mat1, mat2);
    std::cout << "Sum of matrices:" << std::endl;
    sum.print();

    Matrix identity = Matrix::identity(3);
    std::cout << "Identity matrix of size 3:" << std::endl;
    identity.print();

    return 0;
}
```
