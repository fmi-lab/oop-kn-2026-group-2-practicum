# LazyMap

Създайте клас `LazyMap`, който представлява контейнер, резултат от прилагането на функция към всеки елемент от друг контейнер. 
- Класът трябва да приема два шаблонни параметъра: тип на входните елементи и тип на изходните елементи. 
- Конструкторът на `LazyMap` трябва да приема *референция* към контейнер и функция, която да се прилага към всеки елемент от контейнера. 
- Имплементирайте итератор за `LazyMap`, който позволява итерация през резултатите от прилагането на функцията, без да се налага предварително изчисляване на всички резултати. Резултатите трябва да се изчисляват "на място" при итерация. 
- Имплементирайте метод `operator[]`, който позволява достъп до елементите на контейнера по индекс.

## Бонус : 
Имплементирайте итератора на `LazyMap` така, че последователни извиквания за дерефериране на итератора да не водят до повторно изчисляване на резултата, а да се кешира последният изчислен резултат. Т.е, следният код трябва да извърши само едно изчисление на `f(*it)` за всеки елемент на `v`:

```cpp
std::vector<int> v = {1, 2, 3};
auto f = [](int x) { return x * x; };
LazyMap<int, int> lazyMap(v, f);
for (auto it = lazyMap.begin(); it != lazyMap.end(); ++it) {
    std::cout << *it << " " << *it + 1 << " " << *it + 2;
}
```
